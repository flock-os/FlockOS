# Notes About These Modules

## Origin: A Touch of the Gospel

The modules in this folder (`the_gospel_analytics.js`, `the_gospel_apologetics.js`, `the_gospel_certificates.js`, `the_gospel_counseling.js`, `the_gospel_courses.js`, `the_gospel_devotionals.js`, `the_gospel_genealogy.js`, `the_gospel_heart.js`, `the_gospel_journal.js`, `the_gospel_lexicon.js`, `the_gospel_library.js`, `the_gospel_quizzes.js`, `the_gospel_reading.js`, `the_gospel_shared.js`, and `the_gospel_theology.js`) were originally designed as part of **A Touch of the Gospel** — a standalone application concept centered on discipleship, biblical study, and personal spiritual formation tools.

## Not Implemented in FlockOS New Covenant

These modules are **not being implemented** in the current FlockOS New Covenant build. They are retained here for reference and potential future use, but no views, routes, or integrations for these modules exist in `New_Covenant/index.html` or the active script pipeline.

If any of these features are revived, they should be evaluated for alignment with the current New Covenant architecture before being wired in.
